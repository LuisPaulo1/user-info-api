### Tecnologias utilizadas
- Java 11
- Spring (boot, web, validation)
- JPA / Hibernate
- Lombok
- Maven
- JUnit/Mockito


## Como executar a API
### Pré-requisitos
- Java 11

```bash
## clonar repositório
git clone https://github.com/LuisPaulo1/user-info-api.git

## entrar na pasta do projeto user-info-api
cd user-info-api

## executar o projeto
./mvnw spring-boot:run
```
